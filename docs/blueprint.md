# **App Name**: Cecilia Sousa: Harmonia do Olhar

## Core Features:

- Engaging Hero Section: High-impact landing section with a professional portrait, value proposition focused on 'Harmonia do Olhar', and an immediate WhatsApp scheduling CTA.
- Adaptive Dark/Light Mode Toggle: A sophisticated UI control allowing users to switch between a clean light theme and a premium dark theme based on their preference.
- Service Portfolio Showcase: Detailed cards for Brow Lamination, Lash Lifting, and Design de Sobrancelhas highlighting natural results and specialized techniques.
- Interactive Transformation Gallery: A 'Before & After' comparison slider tool that allows potential clients to visualize the subtle but impactful natural transformations.
- Social Proof & Testimonials: Dynamic section displaying verified client feedback and emotional success stories to build trust and authority.
- Strategic Booking Funnel: A multi-step scheduling trigger that prepares the user's intent before redirecting to WhatsApp for personalized conversion.
- AI-Powered Inquiry Assistant: A specialized AI tool that generates personalized message starters for WhatsApp based on user concerns like 'thin eyebrows' or 'fear of artificial looks'.

## Style Guidelines:

- Primary color: Deep Midnight Blue (#0A1E48), symbolizing professionalism, technical mastery, and premium elegance.
- Theme Modes: Light Mode utilizes soft grays and whites for an airy feel; Dark Mode uses the intense Deep Navy (#071025) for a high-end, editorial look.
- Accent color: Sophisticated Silver (#C0C0C0) or Pearl (#FDEEF4) used for interactive elements and subtle highlights against the dark blue base.
- Headline font: 'Belleza' for its humanist sans-serif elegance; Body font: 'Alegreya' for its scholarly and comforting serif readability.
- Minimalist, fine-line icons that emphasize delicacy, precision, and the 'Beauty through technique' philosophy.
- Editorial-style layout with generous negative space, asymmetric balance, and large-scale photography to create a premium magazine feel.
- Smooth, deliberate transitions and parallax scroll effects that mimic the lightness and elegance of the brand's aesthetic.
